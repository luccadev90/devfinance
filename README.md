# DevFinance - Sistema de Finanças Pessoais

Sistema completo para gerenciamento de finanças pessoais com Node.js, Express, MongoDB e Bootstrap.

## 🚀 Tecnologias

- Node.js
- Express
- MongoDB Atlas
- Mongoose
- EJS
- Bootstrap 5

## 📋 Funcionalidades

- ✅ Adicionar transações (receitas/despesas)
- ✅ Status: Pendente ou Pago
- ✅ Editar e excluir transações
- ✅ Alternar status com um clique
- ✅ Filtros por status e tipo
- ✅ Cards com estatísticas em tempo real
- ✅ Exportar dados em JSON
- ✅ Interface responsiva
- ✅ Dados de teste

## 🔧 Como executar

### Pré-requisitos

- Node.js (v14 ou superior)
- MongoDB Atlas (ou MongoDB local)
- NPM ou Yarn
